Le programme

Un parcours en 3 séances progressives

De la découverte à la maîtrise avancée - chaque séance construit sur la précédente pour une montée en compétences rapide.

01
Séance 1 · Fondamentaux
Comprendre & prendre en main Claude IA
Débutants & initiés - Base solide pour bien démarrer
Module 1.1 - Qu'est-ce que l'IA générative ?
Comprendre comment fonctionne Claude, différences avec ChatGPT, Gemini. État actuel de l'IA en 2025-2026.
Module 1.2 - Prise en main de Claude
Interface claude.ai, créer un compte, les différents plans. Premiers prompts en contexte professionnel.
Module 1.3 - L'art du bon prompt
Techniques de prompting : contexte, rôle, format, exemples. Prompts pour la rédaction professionnelle, emails, comptes-rendus.
Module 1.4 - Veille IA : rester à jour
Comment suivre les nouveautés IA sans se perdre. Sources fiables, newsletters, outils de veille recommandés.
02
Séance 2 · Pratique intensive
Automatiser les tâches et la production de contenus
Cas concrets en entreprise - Applicable immédiatement
Module 2.1 - Créer des présentations PowerPoint
Générer un plan de slides, rédiger le contenu, structurer les messages clés. Utiliser Claude + PowerPoint efficacement.
Module 2.2 - Rédiger des rapports professionnels
Rapports d'activité, synthèses exécutives, notes de service, comptes-rendus de réunion. Templates réutilisables.
Module 2.3 - Analyser des données avec Claude
Importer des tableaux, interpréter des chiffres, produire des synthèses chiffrées. Analyse de fichiers Excel/CSV.
Module 2.4 - Automatiser la communication écrite
Emails, propositions commerciales, appels d'offre, fiches produit. Adapter le ton selon l'interlocuteur.
03
Séance 3 · Stratégie & avenir
Maîtriser l'IA et l'intégrer durablement
Passer à l'échelle en entreprise - Vision long terme
Module 3.1 - Workflows IA dans l'entreprise
Identifier les tâches automatisables dans son poste, créer des processus répétables avec Claude. Gain de temps concret.
Module 3.2 - Bonnes pratiques & limites de l'IA
Vérification des contenus générés, confidentialité des données, risques à connaître. Utiliser l'IA de façon responsable.
Module 3.3 - Tendances IA & stratégie d'entreprise
IA agentique, automatisation avancée, outils complémentaires (Gemini, Copilot, Perplexity). Préparer son entreprise.
Module 3.4 - Plan d'action personnel
Chaque participant repart avec son plan d'intégration IA : 3 tâches à automatiser dès la semaine suivante.
